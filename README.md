# Compare Compressed File

Use C# to compare file size between original &amp; compressed tar.gz file.

# Prerequisites

Install [.NET 10.0](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)

# Run

### Open terminal and run the following

Move to directory
```bash
cd CompareCompressedFile
```

Build project
```bash
dotnet build
```

Run Project (Windows)
```bash
dotnet run --project .\CompareCompressedFile\CompareCompressedFile.csproj
```

Run Project (Linux)
```bash
dotnet run --project ./CompareCompressedFile/CompareCompressedFile.csproj
```
